Nosso pricipal Projeto

Pixel Verse

PixelVerse é um mapa interativo baseado em uma arte pixel art de alta qualidade que representa diferentes cidades do mundo (ex.: Tóquio, Paris, Nova York). Cada cidade é dividida em áreas clicáveis, que revelam informações, animações, ou histórias específicas sobre aquele local, como pontos turísticos, gastronomia ou cultura.

Funcionalidades Principais

Mapa Interativo com Pixel Art:

Um mapa desenhado no estilo pixel art detalhado. Regiões do mapa ficam interativas ao passar o mouse (animações ou brilhos destacam as áreas clicáveis). Navegação em Camadas:

Zoom in/out para visualizar diferentes níveis de detalhe (da visão global da cidade até uma rua específica). Integração de mapas 2D estilizados com camadas para guias culturais ou eventos locais. Pontos de Interesse:

Ao clicar em locais como "Torre Eiffel", abre-se uma janela modal com: Curiosidades (ex.: ano de construção, história). Galeria de imagens estilizadas. Links para explorar em tempo real (ex.: tours 360° integrados).

Customização:

Usuários podem criar seus próprios "mapas virtuais" baseados em sua cidade natal ou imaginar uma cidade futurística.

Gamificação:

Usuários ganham conquistas por explorar locais, participar de quizzes culturais ou criar seus próprios guias virtuais.

Modo Noturno/Dinâmico:

O mapa muda dependendo da hora local da cidade (por exemplo, luzes da cidade à noite, céu claro durante o dia). Tecnologias Utilizadas

Front-end:

  • HTML5: Estrutura do mapa interativo.
  • CSS3: Estilizações, animações e transições (como brilhos e sombreamento dinâmico).
  • JavaScript: Interatividade em tempo real (cliques, animações, chamadas AJAX).
  • Canvas API e SVG: Renderização detalhada de elementos do mapa.
  • React.js: Gerenciamento de estado e renderização de componentes modais e dinâmicos.

    • Back-end (para dados dinâmicos):

    Node.js/Express: APIs para carregar informações geográficas e culturais. MongoDB: Banco de dados para armazenar detalhes das cidades e mapas personalizados.

    Extras:

  • WebGL (Three.js): Para animações 3D ou perspectiva de voo ao navegar entre pontos.
  • OpenStreetMap API: Para informações geográficas precisas.

    • Fluxo de Usuário

    Página Inicial:

    Introdução ao projeto com um globo terrestre animado e convite para explorar cidades.

    Exploração:

    Usuário escolhe uma cidade e entra no mapa interativo. Conforme explora, descobre pontos de interesse marcados com animações.

    Interatividade:

    Clicando em pontos, o usuário acessa informações culturais e visuais. Pode deixar comentários ou sugestões para outros exploradores.

    Criação de Mapas:

    Usuários podem usar um editor visual para criar mapas fictícios ou reais. Exemplo de Interface

    Mapa Interativo:

    Cidade representada em pixel art. Áreas interativas piscam suavemente ao passar o mouse.

    Modal de Informações:

    Título, imagem, e breve descrição. Botões para "Ver Galeria", "História" e "Quiz Cultural".

    Barra de Navegação:

    Links para "Explorar Cidades", "Criar Meu Mapa", "Configurações". Impacto e Potencial Este projeto une aprendizado e entretenimento em um formato inovador.
